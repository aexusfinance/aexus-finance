# **Scenario Engine**

The Scenario Engine is a high-fidelity simulation layer capable of processing complex “what-if” prompts.
Users can examine impacts across liquidity, narratives, capital rotation, risk fractures, and behavioral drivers.

Example use cases:

- Market acceleration scenarios
- Liquidity drain events
- Volatility shocks
- Cross-chain capital flows
- Leverage stress scenarios
- Social sentiment cascading
- Narrative momentum forecasting

Example command:

```
aexus scenario "what if the solana ecosystem rallies under a constrained liquidity environment"
```

The engine returns:

- Sentiment posture
- Impact distribution
- Risk nodes
- Behavioral triggers
- Macro-micro reaction chains
- Monitoring points
- Operator progression outcome

The Scenario Engine transforms analysis into a structured, repeatable decision surface.

# **Installation**

## Clone

```
git clone https://github.com/<you>/aexus-cli
cd aexus-cli
```

## Install

```
npm install
```

## Build

```
npm run build
```

## Link globally

```
npm link
```

Activate:

```
aexus
```

# **Environment Variables**

Copy:

